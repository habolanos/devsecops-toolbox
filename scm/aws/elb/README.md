# AWS Elastic Load Balancer Checker

Herramienta para análisis de ALB, NLB y Classic Load Balancers.

## Uso

```bash
python aws_load_balancer_checker.py --profile default --region us-east-1
python aws_load_balancer_checker.py --type alb -o json
```
